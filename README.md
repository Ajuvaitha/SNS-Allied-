# 🌐 SNS College of Allied Health Sciences Website

> 🎓 A modern and responsive website showcasing healthcare education, programs, and institutional excellence.

---

## 📖 About

The **SNS College of Allied Health Sciences Website** is designed to provide a seamless digital experience for students, parents, and visitors. It presents detailed information about the college, its courses, faculty, and events in a structured and user-friendly manner.

This project focuses on delivering a clean UI, smooth navigation, and accessibility across devices.

---

## ✨ Features

* 🏫 About the Institution
* 📚 Courses & Programs
* 📰 Events & Announcements
* 👨‍⚕️ Faculty Information
* 📞 Contact Page
* 🌐 Fully Responsive Design
